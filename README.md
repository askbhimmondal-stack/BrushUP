Brush Up – Web Drawing Tool

Brush Up is a lightweight and easy-to-use web-based drawing application inspired by MS Paint. It allows users to quickly create sketches, drawings, and annotations directly in the browser without installing any software.

The project is designed with a simple, intuitive interface so that anyone can start drawing instantly. Whether you want to doodle, practice digital drawing, explain ideas visually, or create quick illustrations, Brush Up provides a smooth and responsive canvas experience.

✨ Features
🖌️ Freehand Brush Tool – Draw smoothly on the canvas just like MS Paint
🎨 Color Picker – Choose from multiple colors for your drawings
📏 Adjustable Brush Size – Control brush thickness for precise or bold strokes
🧽 Eraser Tool – Easily remove unwanted strokes
📄 Clear Canvas – Start fresh with one click
💾 Save Drawing – Export your artwork as an image file
🖥️ Browser Based – No installation required
⚡ Fast & Lightweight – Runs smoothly on modern browsers
🎯 Use Cases
Quick sketches and doodles
Digital whiteboard notes
Teaching or explaining concepts visually
Practicing drawing digitally
Creating simple graphics
🚀 Tech Stack
HTML5 Canvas
CSS3
JavaScript
📌 Project Goal

The goal of Brush Up is to provide a minimal yet powerful drawing experience on the web, similar to the simplicity of classic MS Paint, while remaining fast, accessible, and easy to use.
